**News Scraper (Python)**

**About**

I built this small project to experiment with extracting real-time news data using Python( *feedparser* & *newspaper3k* ). It pulls articles from RSS feeds and then uses an article parser to get clean, readable content.

It collects useful info like title, author, and publish date and returns everything in a structured format for easy use.

**How to run**

*pip install feedparser newspaper3k*

Then just update the RSS feed URL in the script and run it.
